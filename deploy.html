GitHub Pages
Free static hosting directly from your GitHub repository. Perfect for open source projects and simple deployments.
1
Push your code to a GitHub repository
2
Go to Settings → Pages in your repository
3
Select "Deploy from a branch" and choose main
4
Save and your site will be live in minutes
git add .
git commit -m "Deploy March Madness Bracket"
git push origin main
🌐
Netlify
Powerful static site hosting with continuous deployment, form handling, and serverless functions.
1
Create a Netlify account (free)
2
Click "New site from Git"
3
Connect your GitHub repository
4
Deploy with default settings (build not needed)
# Netlify configuration (netlify.toml)
[build]
  publish = "."
  command = ""
▲
Vercel
Modern frontend deployment with automatic HTTPS, global CDN, and instant rollbacks.
1
Sign up for Vercel (free)
2
Click "New Project" and import from Git
3
Configure framework preset (Other)
4
Deploy and get your live URL instantly
# Vercel configuration (vercel.json)
{
  "version": 2,
  "builds": [
    { "src": "**/*", "use": "@vercel/static" }
  ]
}
