🏀 Google Sheets Integration Setup

Follow these steps to integrate your March Madness Bracket Tracker with Google Sheets for cloud storage and real-time collaboration.

📋 Step 1: Create Google Cloud Project

1 Go to Google Cloud Console
2 Create a new project or select an existing one
3 Enable the Google Sheets API
4 Enable the Google Drive API

🔑 Step 2: Create API Credentials

1 Go to "Credentials" in the left menu
2 Click "Create Credentials" → "API Key"
3 Copy your API key (keep it secure!)
4 Restrict your API key to your domain for security

📊 Step 3: Create Google Sheet Structure

1 Create a new Google Sheet named "March Madness 2026"
2 Create the following sheets:
  • Brackets - Store user bracket data
  • Results - Store tournament results
  • Users - Store user information
  • Settings - Store configuration

📝 Step 4: Set Up Sheet Columns

Brackets Sheet:

A: UserName | B: Round1_Picks | C: Round2_Picks | D: Round3_Picks | E: Round4_Picks | F: Round5_Picks | G: Round6_Picks | H: Score | I: Submitted | J: Timestamp

Results Sheet:

A: Round | B: MatchupID | C: Team1 | D: Team2 | E: Winner | F: Timestamp

Users Sheet:

A: UserName | B: Email | C: Created | D: LastActive

🔗 Step 5: Get Sheet ID

1 Open your Google Sheet
2 Look at the URL: https://docs.google.com/spreadsheets/d/[SHEET_ID]/edit
3 Copy the [SHEET_ID] part

⚙️ Step 6: Configure the Application

1 Open the bracket application
2 Enter your API key and Sheet ID when prompted
3 Test the connection
⚠️ Security Notice: Never expose your API key in client-side code in production. Use a backend proxy or serverless function for API calls.
✅ Benefits: Once set up, you'll have cloud storage, real-time collaboration, automatic backups, and easy data export!

🚀 Alternative: Use Web App URL

For easier setup without API keys, you can deploy as a Google Apps Script Web App:

1 Go to Google Apps Script
2 Create a new project and paste the provided script
3 Deploy as Web App
4 Use the Web App URL in your bracket application
